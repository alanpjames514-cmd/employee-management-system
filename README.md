# Employee Management System

## 📌 Project Description
This is a console-based Employee Management System built using Python and SQLite. The system allows users to login as Admin, Manager, or Employee and perform operations based on their roles.

---

## 🚀 Features
- User Authentication (Login System)
- Role-Based Access (Admin, Manager, Employee)
- Add Employee
- View Employee Details
- SQLite Database Integration

---

## 🗄️ Database
The system uses SQLite database to store user and employee data.

---

## ▶️ How to Run
1. Install Python
2. Open terminal
3. Navigate to project folder
4. Run:

5. python main.py
